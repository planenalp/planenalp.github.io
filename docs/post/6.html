小细节

  • 要是图片排版乱七八糟可以在每串图片代码前一行加个 # 强行分段
  • 可以直接只用 width = "300"，不需 height = "300"，会自动等比缩放，就像只用 width = "50%" 一样
  • HTML 标签 + base64 无效
  • 可用 p 或类似字符如 abc 代替 div

相当于 [](url)，意义不大，可能默认兼容fancybox？

或

<img src="URL" width = "50%" align=center alt="name" />
